reverse 是一个内置函数，它可以反转一个列表：

    (reverse '(a b c d))
    => '(d c b a)

这是一个简单的实现

```scheme
(define (reverse l)
  (cond
    ((null? l) '())
    (else (append (reverse (cdr l))
                  (list (car l))))))
```

代入参数 '(a b c) 推导一下：

```
(reverse '(a b c))

(append (reverse '(b c))
        '(a))

(append (append (reverse '(c))
                '(b))
        '(a))

(append (append (append (reverse '())
                        '(c))
                '(b))
        '(a))

(append (append (append '() '(c))
                '(b))
        '(a))

(append (append '(c) '(b))
        '(a))

(append '(c b) '(a))

=> '(c b a)
